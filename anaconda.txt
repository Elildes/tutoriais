##### INSTALAR O ANACONDA NAVIGATOR NO LINUX #####

1. Baixar o Miniconda (conda + Python):
> https://docs.conda.io/en/latest/miniconda.html

2. Instalar o Miniconda:
> bash <nome_do_arquivo.sh>

3. Atualizar o Miniconda:
> conda update conda

4. Baixar/instalar o Anaconda Navigator (Modo grÃ¡fico)
> conda install -c anaconda anaconda-navigator, ou:

4. Baixar o arquivo.sh do site:
> https://www.anaconda.com/distribution/#linux

4. Instalar o Anaconda navigator:
> sudo bash nome_do_arquivo.sh

Obs.: caso queira atualizar digite o comando:
> bash nome_do_arquivo.sh -u

5. Abrir o Anaconda Navigator:
> <abrir o terminal>
> anaconda-navigator
